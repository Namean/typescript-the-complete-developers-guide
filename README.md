### Sort



The goal of this project is to take each of these different data structures and sort them
from least to greatest.


Array of numbers [10, 5, 18, -3] ---> [-3, 5, 10, 18]
String 'PoaJB' ---> 'aBJoP'
LinkedList [10->-3->27->5] ---> [-3->5->10->27]
